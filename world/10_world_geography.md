# 異世界の地理・構造

## 現実世界（東京）— 序盤のみ

- 蓮が暮らしていた世界。普通のサラリーマン生活
- 通勤中に刀に触れた瞬間、キラーが創造した異世界に飛ばされる
- **本編の舞台ではない**（序盤の導入のみ）
- 戻れるかどうかは不明（蓮にとっての動機の一つ）

---

## キラーが創造した異世界——「エルダ」

キラーが創造主Aから奪った力で創り出した世界。蓮が刀に触れた瞬間に発動した。

### 世界の全体像

大陸一つの世界。北半球に天使の領域、南半球に悪魔の領域、中央帯に人間が暮らす。

```
    ┌────────────────────────────────────┐
    │         天使の領域「聖域」          │  北方
    │    （セラフ・ドミニオン）           │
    │  秩序・管理・完璧な都市群          │
    ├────────────────────────────────────┤
    │       天使派領土                    │
    │  聖都エクレシア / 前線都市ライン    │
    ├ ─ ─ ─ ─ 北部戦線 ─ ─ ─ ─ ─ ─ ─ ─ ┤
    │       灰色地帯（グレイランド）      │  ← 主戦場
    │  廃墟都市 / 流動する前線            │
    ├ ─ ─ ─ ─ 南部戦線 ─ ─ ─ ─ ─ ─ ─ ─ ┤
    │       悪魔派領土                    │
    │  自由都市ラグナ / 港町群            │
    ├────────────────────────────────────┤
    │         悪魔の領域「魔界」          │  南方

    ※中央帯の東端に「市場」（独立派拠点）
    ※世界の天頂に「裂け目」（創造主たちの領域への接点）
```

## 天使の領域「聖域（セラフ・ドミニオン）」

### 地理
- 世界の北方に広がる白銀と光の領域
- 整然とした都市が幾何学的に配置されている
- 天候は常に晴れ。気温・湿度は「最適値」に固定
- 自然すら管理されている——木々の高さ、花の配色、川の流量

### 特徴
- 天使のみが本来の住人
- 人間は「教会」所属でも聖域には入れない（上位天使の許可が必要）
- 美しく完璧だが、息苦しい——個人の意志が入り込む余地がない
- **蓮はArc 4で初めて足を踏み入れる**

### 主要拠点
- **天使議会場「アドナイ」** — セラフィム（消息不明）の空席がある。大天使が運営
- **ミカエルの砦「ソードゲート」** — 人間界との境界を監視する軍事拠点

---

## 悪魔の領域「魔界（パンデモニウム）」

### 地理
- 世界の南方に広がる炎と闇の領域
- 統一的な構造はなく、七大悪魔がそれぞれの領地を持つ
- 気候は領主の性質に左右される（アスモデウスの領地は常に嵐）
- 危険だが、活気と刺激に満ちている

### 特徴
- 弱い者は生き残れない。強い者には無限の可能性
- ルールはない。あるのは力の序列のみ
- 人間でも力があれば認められる（天使の領域との最大の違い）

### 主要拠点
- **七王座** — 七大悪魔の居城。それぞれ異なる環境
- **アスモデウスの城「嵐の玉座」** — 常に暴風が吹き荒れる山頂の城

---

## 人間の領域「中央帯」

天使の領域と悪魔の領域に挟まれた帯状の地域。人間が暮らし、代理戦争の舞台となる。

### 天使派領土（北部）

#### 聖都エクレシア
- 天使派の首都。大聖堂を中心とした城塞都市
- 教会の司令部が置かれている
- 秩序正しい街並み。市民は天使の加護の下で安定した生活を送る
- ただし自由は制限される——行動規範、外出制限、情報統制
- **ロイがミカエルと契約した場所**
- 人口：約8万人

#### 前線都市ライン
- 北部戦線に沿って並ぶ3つの要塞都市
- **リヒト（西）** — 防衛特化。城壁が三重。市民の避難所
- **クロイツ（中央）** — 教会軍の主力駐屯地。最大の軍事拠点
- **ゼーレ（東）** — 情報・諜報の拠点。ガブリエル系の契約者が多い

### 灰色地帯（グレイランド）— 主戦場

- 中央帯の中心。天使派にも悪魔派にも属さない緩衝地帯
- **代理戦争の最前線**。常に戦闘が発生している
- 廃墟化した都市、焼け野原、壊れた結界の残骸が散在
- 前線は流動的——一日で数km動くこともある
- 一般市民はいない（かつての住民は南北に避難済み）

#### 廃都グラウ
- かつては中央帯最大の都市だった
- 大規模戦闘で壊滅し、現在は廃墟
- 両軍にとって戦略的価値が高い（中央帯の交差点）
- **Arc 3の大規模戦闘の舞台**

#### 裂け谷（クレバス）
- 灰色地帯の東端に走る巨大な地割れ
- 世界の創造時に生じた歪み
- 次元が不安定で、創造主たちの領域への接点が現れることがある
- **Arc 7で重要な場所になる**

### 悪魔派領土（南部）

#### 自由都市ラグナ
- 悪魔派の最大都市。しかし「首都」ではない（悪魔派に首都の概念はない）
- 闘技場、賭博場、市場が密集した雑然とした街
- 力があれば誰でも歓迎される。なければ搾取される
- 情報と物資の集積地
- 人口：約6万人（流動的）

#### 港町群（サウスコースト）
- 南部海岸線に点在する小さな港町の集まり
- 悪魔の領域との交易路
- 密輸、傭兵の募集、非合法取引の温床
- 独立派とも繋がりがある

### 独立派の拠点

#### 市場（マルクト）
- 中央帯の東端、裂け谷の手前に位置する
- 崖の中腹を削り出した隠れ里のような街
- **蓮が最初に身を寄せる場所**
- 元天使派・元悪魔派・傭兵・商人・難民が集まる
- 情報と物資の「取引」で成り立つ中立地帯
- 人口：約3,000人
- リーダー：**ヴェルナー**（元天使派の老練な戦士。教会に幻滅して離脱）
- 天使派も悪魔派も市場の存在を知っているが、潰す価値がないため黙認

## 戦線の現状（物語開始時点）

### 北部戦線
- 教会軍（天使派）が守勢に回っている
- 悪魔派の突撃隊が頻繁に攻勢をかける
- 膠着状態が続いており、大きな変動はない
- ミカエルが直接介入すれば戦局は一変するが、「人間の代理戦争」の原則を守っている

### 南部戦線
- ギルド（悪魔派）の防衛線
- 天使派の浸透工作（スパイ、暗殺）が主な脅威
- 大規模戦闘よりも暗闘・諜報戦が多い

### 灰色地帯
- 両軍の小部隊が散発的に衝突
- 廃都グラウの支配権を巡って、月に数回の中規模戦闘
- **Arc 3で大規模戦闘が発生し、灰色地帯の均衡が崩れる**

---

## 戦争の構図

```
            天使の領域
                ↓ 指令
            教会（天使派人間）
                ↕ 北部戦線
        灰色地帯（主戦場）← 独立派は介入を避ける
                ↕ 南部戦線
            ギルド（悪魔派人間）
                ↑ 支援
            悪魔の領域
```

### なぜ天使と悪魔は直接戦わないのか
- 天使と悪魔の全面戦争は、世界そのものの崩壊を招く
- キラーが創った世界は創造主Aの力の残滓で維持されている——上位存在の全力戦闘は世界の基盤を壊す
- 人間を使う「代理戦争」は、世界を壊さずに勢力争いをする唯一の手段
- **ただし、アスモデウスがキラーの刀を手にしたArc 5以降、この均衡は崩れ始める**

---

## 創造主たちの領域

- 世界の「上位」に存在する超次元空間。複数の創造主が存在する
- 通常は到達不可能（マーキングがあれば可能）
- キラーが創造した異世界は、この領域に通じるように設計されている（創造主Bへの接近手段）
- 裂け谷の最深部に「門」がある
- 時間と空間の概念が異なる
- 創造主Aの残留思念が漂っている
- 美しくも寂しい空間——創造主Aの孤独が形になったような場所
- Arc 7-8の舞台

## 蓮の移動経路（アーク対応）

| Arc | 主な舞台 | 蓮の位置 |
|-----|---------|---------|
| 1 | 東京 → 灰色地帯 → 市場 | 異世界に飛ばされ、凛に保護される |
| 2 | 市場 / 灰色地帯（周辺） | 独立派で修行。戦線偵察 |
| 3 | 廃都グラウ（灰色地帯） | 大規模戦闘に巻き込まれる |
| 4 | 聖都エクレシア → 天使の領域 | ミカエルに召喚。聖域を見る |
| 5 | 境界地帯 → 悪魔派領土 | 凛の重傷、ロイの裏切り、刀の喪失 |
| 6 | 自由都市ラグナ → 灰色地帯 | 刀なしで成長。アスモデウスと対峙 |
| 7 | 裂け谷 → 創造主たちの領域 | 真実の開示。キラーの正体 |
| 8 | 全域 | 最終決戦 |
