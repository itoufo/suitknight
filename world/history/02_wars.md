## 天魔戦争（上位存在の直接衝突）

### 第一次天魔戦争（Y5-10）

| 項目 | 内容 |
|------|------|
| 期間 | 約5年 |
| 天使側 | ミカエル指揮。ガブリエル・ラファエル・ウリエルが参戦 |
| 悪魔側 | アスモデウス指揮。サタン・ベルゼブブが前線 |
| 舞台 | 中央帯全域 |
| 結果 | 引き分け。双方が世界崩壊の危険を認識し停戦 |

**戦略的教訓**:
- 上位存在同士の戦闘は世界の基盤を損傷する
- 「勝っても世界が壊れては意味がない」——代理戦争への最初の気づき
- 蓮の分析:「核抑止に似ている。MAD（相互確証破壊）の原始形」

### 第二次天魔戦争（Y10-25）

| 項目 | 内容 |
|------|------|
| 期間 | 約15年 |
| 天使側 | ミカエル指揮。初めて人間部隊を組織的に投入 |
| 悪魔側 | アスモデウス・ベルフェゴールの二頭作戦 |
| 舞台 | 中央帯北部〜南部 |
| 兵力 | 天使派人間：約3,000名 / 悪魔派傭兵：約1,500名 |
| 結果 | 天使派の辛勝。南部の一部を確保 |

**主要会戦**:
- **グラウ平原の戦い（Y18）**: 天使派人間部隊の初めての大規模戦闘。契約者の力で押すが、悪魔派傭兵の個人技に苦戦。双方合わせて約800名が死亡
- **南部港攻防戦（Y23）**: 天使派が悪魔の補給路を断つ試み。成功するが、維持できず撤退

**戦略的教訓**:
- 人間の組織力は有効だが、個人の突出した力には対抗しにくい
- 補給線の確保が戦争の帰趨を決める
- 蓮の分析:「兵力の質と量のトレードオフ。教会はWW1のドイツ軍に似ている——組織で勝つが柔軟性に欠ける」

### 第三次天魔戦争（Y25-47）

| 項目 | 内容 |
|------|------|
| 期間 | 約22年 |
| 天使側 | ミカエル指揮。人間部隊が主力に成長 |
| 悪魔側 | アスモデウス・サタンの攻勢連合 |
| 舞台 | エルダ全域 |
| 兵力 | 天使派：約10,000名 / 悪魔派：約6,000名 |
| 結果 | 「大裂け」事件で強制停戦 |

**主要会戦**:
- **グラウ攻城戦（Y30-32）**: グラウを巡る2年間の攻防。都市は壊滅的被害を受ける
- **北部回廊の戦い（Y38）**: 天使派が悪魔派の北進を阻止。リヒトの原型となる防衛陣地が構築される
- **「大裂け」直前の決戦（Y47）**: ミカエルとアスモデウスの再直接対決。世界が裂けた

**戦略的教訓**:
- 長期戦は双方を消耗させ、エスカレーションを招く
- 上位存在の介入は戦場だけでなく世界構造そのものを破壊する
- 蓮の分析:「WW1の塹壕戦と同じ構図。膠着→エスカレーション→破局。教訓が活かされていない」

## 代理戦争時代の主要会戦

### ハルム河の戦い（Y85頃）

| 項目 | 内容 |
|------|------|
| 天使派 | 初期教会軍・約2,000名。指揮官不明 |
| 悪魔派 | 初期ギルド・約800名。伝説的戦将が指揮 |
| 舞台 | 灰色地帯中央のハルム河（後のグレンツ河の支流） |
| 結果 | ギルドの圧勝。教会軍は半数を失う |

- 10倍の兵力差を欺瞞作戦で覆した伝説的会戦
- ギルドの戦将が偽の渡河地点を設定し、教会軍の主力を分散させた
- 分散した教会軍を各個撃破
- ※戦将の詳細は `world/history/04_historical_figures.md` 参照
- 蓮の分析:「ナポレオンの内線作戦の教科書的実践。少数が多数に勝つ定石」

### クロイツ防衛戦（Y120頃）

| 項目 | 内容 |
|------|------|
| 天使派 | 教会軍・約5,000名。要塞防衛の名将が指揮 |
| 悪魔派 | ギルド連合・約3,000名 |
| 舞台 | 建設中のクロイツ要塞 |
| 結果 | 天使派の防衛成功。ギルドは撤退 |

- 教会の要塞防衛ドクトリンが確立された会戦
- 三重防衛線の原型が実戦で証明された
- ※指揮官の詳細は `world/history/04_historical_figures.md` 参照
- 蓮の分析:「マジノ線の成功版。ただし、防衛に成功しても戦争は終わらない」

### 灰色地帯の「100日戦争」（Y210頃）

| 項目 | 内容 |
|------|------|
| 天使派 | 教会軍・約8,000名 |
| 悪魔派 | ギルド・約4,000名 |
| 舞台 | 灰色地帯全域 |
| 結果 | 双方消耗して原状回復。100日で前線が元に戻る |

- 100年の膠着期における最大規模の攻勢
- 天使派が北部から大規模進撃を試みたが、補給線が延びすぎて頓挫
- ギルドの反撃も教会の要塞線で止まる
- 蓮の分析:「ナポレオンのロシア遠征。補給の限界を超えた進撃は必ず破綻する」

### 独立派粛清作戦（Y170頃）

| 項目 | 内容 |
|------|------|
| 教会軍 | 粛清部隊・約500名。ウリエルの指示 |
| 独立派 | 初期独立運動・約200名 |
| 舞台 | 中央帯東部 |
| 結果 | 独立派壊滅。指導者暗殺 |

- 教会が独立運動を武力で鎮圧した記録
- 教会の公式記録からは削除されている（オットーが発見）
- ウリエルが人間の粛清部隊を直接指揮した数少ない事例
- 蓮の分析:「天使＝善ではない。秩序のために反対勢力を殲滅する——帝国主義の論理だ」

## 戦争記録の総括

### 共通パターン（蓮が抽出）
1. **膠着→エスカレーション→破局→停戦→膠着**のサイクルが繰り返される
2. 補給線を確保した側が最終的に有利
3. 少数精鋭は短期決戦で有効だが、持久戦では数に押される
4. 上位存在の介入は戦場ではなく世界構造を破壊する
5. 戦争を終結させようとした全ての試みが失敗している

### 蓮が参照する教訓
- 孫子「戦わずして勝つ」= 戦闘回避の戦略的正当化
- ナポレオンの内線作戦 = 少数精鋭で大軍に対抗する方法
- WW2パルチザン = 灰色地帯でのゲリラ戦の参考
- 補給線の重要性 = どの時代も変わらない戦争の基本
