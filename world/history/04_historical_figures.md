# 歴史的人物

> フレーム: オットーの書庫に保管された人物記録。蓮が歴史書で学び、実戦で参照する対象。

---

## Baldur（バルドゥル）— 鋼壁の将軍

> 教会の要塞防衛を確立した将軍。エルダのクラウゼヴィッツ。

### 基本情報

| 項目 | 内容 |
|------|------|
| 活動時期 | Y100-140頃 |
| 所属 | 天使派・教会軍 |
| 役職 | 教会軍総指揮官 |
| 功績 | 三要塞都市（リヒト・クロイツ・ゼーレ）の防衛体系を設計・構築 |

### 人物像
- 第三次天魔戦争の生き残り世代。戦争の破壊を子供時代に目撃した
- 「二度と前線を崩させない」という信念で、教会軍の防衛ドクトリンを構築
- 三重防衛線の概念を考案：第一線（警戒）→第二線（主抵抗）→第三線（予備）
- 戦略思想：「攻める必要はない。守りきれば、敵は自ら崩れる」
- クロイツ防衛戦（Y120頃）で自ら指揮を取り、ギルド連合3,000名の攻勢を撃退

### 遺産
- 現在のアラリックの要塞防衛ドクトリンはバルドゥルの理論を継承したもの
- 「バルドゥルの三層防衛」は教会軍の基本教範として100年以上使われている
- 弱点もバルドゥルから受け継がれている——攻勢能力の欠如

### 蓮の分析
> 「クラウゼヴィッツの『防御は攻撃より強い戦闘形態である』を体現した人。でも、防御だけでは戦争は終わらない。マジノ線の教訓だ」

## Sigrid（ジークリート）— 欺瞞の戦将

> 10倍の敵を欺瞞で破ったギルドの伝説的戦将。

### 基本情報

| 項目 | 内容 |
|------|------|
| 活動時期 | Y80-110頃 |
| 所属 | 悪魔派・ギルド |
| 役職 | 傭兵団長→ギルド連合軍指揮官 |
| 功績 | ハルム河の戦い（Y85頃）で10倍の教会軍を撃破 |

### 人物像
- ギルド史上最も有名な戦術家。「嘘を本当にする」のが得意だった
- 少数の傭兵で大軍に対抗するため、欺瞞作戦を極限まで洗練させた
- ハルム河の戦いでは、偽の渡河地点を5箇所設定し、教会軍の主力を分散させた後、各個撃破
- 戦略思想：「敵が見ているものが真実とは限らない。そして、敵が見ていないものこそ真実だ」
- マモンの経済網を利用して偽情報を流す手法を開発

### 遺産
- フェンリスの欺瞞戦術の直接的な先祖
- ギルドの「少数精鋭・欺瞞重視」の戦闘文化を確立
- 欠点もフェンリスに受け継がれている——持久戦での脆弱さ

### 蓮の分析
> 「ナポレオンのアウステルリッツ。敵の思い込みを利用して、自分の望む場所で戦う。孫子の『虚実』だ。ただ、勝ち続けることはできない——補給が続かないから」

## Hagen（ハーゲン）— 第三の旗

> 勢力統一を試み暗殺された独立派指導者。ヴェルナーの前任者的存在。

### 基本情報

| 項目 | 内容 |
|------|------|
| 活動時期 | Y150-170頃 |
| 所属 | 独立派（第一次独立運動） |
| 役職 | 独立運動指導者 |
| 最期 | Y170頃、教会の粛清部隊により暗殺 |

### 人物像
- 元教会の修道兵。代理戦争の本質に気づき、「第三の選択肢」を掲げた
- 中央帯の民衆を組織し、教会にもギルドにも属さない勢力の構築を目指した
- カリスマ的な指導力を持ち、最盛期には約2,000名の支持者を集めた
- 教会内の改革派と接触し、内部からの変革を試みた
- しかし改革派がウリエルに密告。ハーゲンは粛清対象に
- ウリエル直属の粛清部隊に追い詰められ、暗殺された

### 遺産
- 独立運動はハーゲンの死後30年以上復活できなかった
- しかしハーゲンの思想は密かに受け継がれ、後のマルクト建設の精神的基盤に
- ヴェルナーはハーゲンの理念を知った上で、独立派を再建した
- オットーが教会書庫でハーゲンに関する記録を発見したことが、オットーの粛清対象となった原因の一つ

### 蓮の分析
> 「チェ・ゲバラとガンジーを足して割ったような人。理想は正しかったけど、組織の壁を甘く見た。教会内部の改革派を信じたのが致命的——組織のインセンティブ構造を理解していなかった」

## Marguerite（マルグリット）— 失われた和平

> 戦争終結を最も近くまで達成した人物。

### 基本情報

| 項目 | 内容 |
|------|------|
| 活動時期 | Y195-205頃 |
| 所属 | ギルド（情報屋） |
| 役職 | 非公式外交官 |
| 最期 | Y205頃に姿を消す。消息不明 |

### 人物像
- ギルドの情報屋として教会・ギルド双方に人脈を持った希有な人物
- 3年間にわたる非公式和平交渉を主導
- 教会の穏健派とギルドの現実主義者を結びつけ、停戦合意の草案まで到達
- しかしマモンが「戦争終結は経済的損失」と判断し、交渉網を経済的圧力で破壊
- マルグリットは姿を消した。殺されたのか逃亡したのか、記録は残っていない

### 遺産
- 「和平は不可能」という現在の常識を作った失敗
- しかしマルグリットの交渉記録は、オットーの書庫に保管されている
- 蓮がこの記録を読み、「なぜ失敗したのか」を分析することが、蓮の独自戦略の出発点になる

### 蓮の分析
> 「マルグリットは交渉の天才だったけど、ステークホルダー分析が甘かった。マモンの利害構造を読み切れなかった。戦争を終わらせるなら、戦争で利益を得ている全員の代替利益を設計しないといけない——IT的に言えば、全ステークホルダーの要件定義だ」

## 歴史的人物と現代キャラクターの対応
