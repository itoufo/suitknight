# 世界の基本法則

## キラーの吸収能力

通常、力の移転は不可能。ただし一つの例外が存在する。

- **キラー**は殺した相手の力を吸収できる
- これは三界の法則を超えた異常能力
- 創造主Aすら殺したキラーの本質的な力
- 現在は刀に封じられており、0.1秒の支配中にのみ発動可能
- この能力の存在は、ほとんどの存在が知らない

---

## 起源事件

### 創造主たちと三界の創造

創造主は複数存在し、各創造主が1つの世界を持つ。物語に関わるのは主に創造主A・B・Cの三者。

1. **人間界** — 物質の世界。最も脆く、最も多様な生命が息づく
2. **天使悪魔界** — 霊的な世界。秩序と自由の二つの領域からなる
3. **創造主たちの領域** — 三界の「上位」に存在する超次元空間。複数の創造主が存在する

### 創造主Aの死

- 創造主Aは、自ら創った最初の存在「キラー」（原初の存在）に殺された
- 創造主Aは抵抗せず殺された（理由は物語終盤で明かす）
- キラーが創造主Aを殺した後、**別の創造主（創造主B）がキラーを刀に封じた**
- マーキングは事前に存在したのではなく、後に人間が刀に触れた際に発生した
  - 刀の姿では使えなかった創造主Aの力が、人の手に触れ「1人格」として処理されたことで発動
- 創造主Aの死後、天使と悪魔は指導者を失い、互いに争い始めた

## 力の序列

| 階層 | 存在 | 補足 |
|------|------|------|
| S+++ | 創造主たち | 創造主A死亡（残存力のみ）。創造主Bは生存 |
| S++ | キラー（完全解放時） | 現在は封印中 |
| S+ | 熾天使（セラフィム） | 3体のみ。消息不明 |
| S | 大天使 / 魔王級（七大悪魔） | ミカエル、アスモデウス等 |
| A | 上位天使 / 上位悪魔 | 権天使、力天使、大悪魔 |
| B | 中位天使 / 中位悪魔 | 数百体 |
| C | 下位天使 / 下位悪魔 | 数千体 |
| D | 力を持つ人間（上位） | リン、ロイなど |
| E | 力を持つ人間（一般） | 主人公（通常時） |
| F | 一般人 | 力を持たない |

※ 主人公の0.1秒抜刀時はS級相当の瞬間的破壊力

## 力の測定方法

### オーラの密度
- オーラの濃さ＝基礎戦闘力
- 密度が高いほど攻撃力・防御力が上がる
- 天使はオーラが白〜金、悪魔は赤〜黒が多い

### 領域の広さ
- 領域を展開できる範囲＝支配力の指標
- 大天使/魔王級は数キロメートル規模の領域を展開可能
- 人間は数メートル〜数十メートルが限界

### 武器の覚醒段階
- 初期形態 → 第一解放 → 第二解放 → 完全解放
- 段階が高いほど強力

## 「例外」の存在

### マーキング保持者

- 刀（キラー）に触れた人間に刻まれた創造主Aの力の断片
- 蓮が刀に触れた際、「1人格」として処理され世界創造と同時にマーキングが発生
- 保持者は通常の力の序列から外れることがある
- **主人公・鷹野蓮はマーキング保持者**（本人は知らない）
- マーキングの影響：オーラの色が定まらない、力の成長パターンが異常

### キラーの吸収能力

- 殺した相手の力を自分のものにできる
- 三界の法則を超えた唯一の例外
- この能力があるからこそ、キラーは創造主Aを殺せた
