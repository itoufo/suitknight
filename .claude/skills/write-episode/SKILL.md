# エピソード執筆スキル

## 役割
スーツナイトの各話を、設定の一貫性を保ちながら、読者を引きつける構成で執筆する。

## 起動コマンド

```
/write-episode [話数] [アーク番号]
```

例: `/write-episode 5 2` → 第5話（Arc 2内）を執筆

---

## アーク→話数の分割原則

### 基本ルール
- **1アーク = 3〜6話**（1アーク1話は禁止）
- 各話に**明確な引き（クリフハンガー）**を設ける
- 各話に**最低1つのフック要素**を含める
- アーク冒頭の話は**世界への没入**を優先
- アーク末尾の話は**次アークへの橋渡し**を含める

### フック要素の種類

| 種類 | 説明 | 例 |
|------|------|-----|
| 謎の提示 | 読者に「なぜ？」を与える | オーラの色が定まらない |
| 危機の予兆 | 不穏な空気を残す | キラーの声色の変化 |
| 意味深な台詞 | 裏の意味を持つ発言 | 上司の「気をつけろよ」 |
| 感情の揺さぶり | キャラクターの感情で読者を引く | 蓮の「抜かない」覚悟 |
| 行動の衝撃 | 予想外の展開 | 突然の異世界転移 |
| クリフハンガー | シーンの途中で切る | 戦闘開始直前で次話 |

### 引きの強度基準

| 強度 | 基準 | 使いどころ |
|------|------|-----------|
| ★★★★★ | 読者が即座に次話を開く | アーク冒頭、クライマックス前 |
| ★★★★ | 強い続きへの期待 | 重要シーンの前後 |
| ★★★ | 穏やかだが気になる | 日常パート後、訓練パート後 |
| ★★ | 余韻型 | アーク最終話（次アークへ繋ぐ） |

## 文体ルール

### 視点
- **一人称主観**（蓮の視点）
- 神視点は禁止
- 蓮が知らない情報は書かない
- 蓮が気づかないことは読者にも気づかせない（ただし描写で伏線は置ける）

### 文体の特徴
- **簡潔で乾いた文体**（蓮の性格を反映）
- 内面描写は控えめ（感情を語りすぎない）
- 五感描写を重視（匂い、音、触感、温度）
- 比喩はサラリーマン視点のものを使う（「納期のように迫る」「残業より長い夜」等）
- ダッシュ（——）を効果的に使う
- 一文は短め。長文は緊迫感を損なう

### キラーの台詞
- **太字**で表記
- 頭の中に直接響く描写
- 嘲笑と挑発が基本トーン
- 「抜け」の使い方にバリエーションを持たせる

### 戦闘描写
- 速度感を重視（短文の連続）
- 痛みの具体的描写（打撲、切傷、筋肉の悲鳴）
- 環境の活用を細かく描く
- スーツの素材感を戦闘に活かす

## セルフチェックリスト

### 禁則チェック（❌が一つでもあれば書き直し）

#### 世界観
- [ ] 天使を純粋な善として描いていないか
- [ ] 悪魔を純粋な悪として描いていないか
- [ ] 「正義vs悪」の構図になっていないか

#### 主人公
- [ ] 刀を気軽に抜いていないか（各アーク0-2回上限）
- [ ] サラリーマンの感覚を失っていないか
- [ ] 英雄的に振る舞っていないか
- [ ] 説教していないか
- [ ] 「選ばれた理由」を探していないか

#### 物語
- [ ] 神視点を使っていないか
- [ ] パワーインフレが起きていないか
- [ ] ご都合主義の勝利になっていないか

#### 0.1秒ルール
- [ ] 0.1秒を延長していないか
- [ ] キラーの支配を弱めていないか
- [ ] 抜刀のリスクを軽減していないか
- [ ] 0.1秒を使いすぎていないか

### 品質チェック

#### フック
- [ ] 冒頭30行以内に読者の注意を引く要素があるか
- [ ] 末尾に次話への引きがあるか
- [ ] 途中に最低1つの「謎」または「緊張」があるか

#### 感情曲線
- [ ] 感情の起伏が1話の中にあるか（ずっと同じテンションは禁止）
- [ ] 感情曲線設計（emotional_curves.md）と整合しているか

#### 伏線
- [ ] 該当話で配置すべき伏線が配置されているか
- [ ] 早期に開示してはいけない情報を出していないか

#### 文体
- [ ] 一人称主観が維持されているか
- [ ] 五感描写が含まれているか
- [ ] キャラクターの口調が設定通りか

## 作者裏話・裏設定セクション（必須）

各話の末尾に必ず「作者裏話・裏設定」セクションを設ける。読者のエンゲージメントを高め、考察を促す重要な要素。

### 記載ルール

- **最低3項目、最大5項目**
- 各項目に**太字の見出し**をつける
- 以下の5種類からバランスよく選ぶ：

| 種類 | 説明 | 例 |
|------|------|-----|
| 執筆意図 | なぜその描写・展開にしたか | タイトルの由来、シーン構成の理由 |
| 小ネタ解説 | 本文中の細かい描写の意味 | 缶コーヒー3本＝蓮の合理性 |
| 裏設定ほのめかし | 本編で明かしていない設定をチラ見せ | 「接触の瞬間に実はとんでもないことが…」 |
| キャラクター掘り下げ | キャラの行動原理や内面の補足 | キラーの第一声が「抜け」である理由 |
| 伏線ヒント | 読者に考察を促す匂わせ | 「この呼び方の響きが変わっていきます」 |

### 禁則（裏話セクション）

- ❌ 読者知識管理で「絶対に早出ししてはいけない情報」を明かさない
- ❌ 伏線の答えを直接言わない（匂わせまで）
- ❌ 今後の展開を具体的にネタバレしない
- ⭕ 「先の話で明かされます」程度の予告はOK
- ⭕ 設定の「存在」を示唆するのはOK（内容を明かさない）
- ⭕ 執筆上の苦労や楽しさを語るのはOK

### トーン

- 作者として読者に語りかける口調（本文の一人称とは別）
- 親しみやすく、でも出し惜しみする
- 「お楽しみに」「気づいた方は鋭い」など、考察を煽る表現を使っていい
