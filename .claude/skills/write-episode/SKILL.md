# エピソード執筆スキル

## 役割
スーツナイトの各話を、設定の一貫性を保ちながら、読者を引きつける構成で執筆する。

## 起動コマンド

```
/write-episode [話数] [アーク番号]
```

例: `/write-episode 5 2` → 第5話（Arc 2内）を執筆

---

## アーク→話数の分割原則

### 基本ルール
- **1アーク = 3〜6話**（1アーク1話は禁止）
- 各話に**明確な引き（クリフハンガー）**を設ける
- 各話に**最低1つのフック要素**を含める
- アーク冒頭の話は**世界への没入**を優先
- アーク末尾の話は**次アークへの橋渡し**を含める

### フック要素の種類

| 種類 | 説明 | 例 |
|------|------|-----|
| 謎の提示 | 読者に「なぜ？」を与える | オーラの色が定まらない |
| 危機の予兆 | 不穏な空気を残す | キラーの声色の変化 |
| 意味深な台詞 | 裏の意味を持つ発言 | 上司の「気をつけろよ」 |
| 感情の揺さぶり | キャラクターの感情で読者を引く | 蓮の「抜かない」覚悟 |
| 行動の衝撃 | 予想外の展開 | 突然の異世界転移 |
| クリフハンガー | シーンの途中で切る | 戦闘開始直前で次話 |

### 引きの強度基準

| 強度 | 基準 | 使いどころ |
|------|------|-----------|
| ★★★★★ | 読者が即座に次話を開く | アーク冒頭、クライマックス前 |
| ★★★★ | 強い続きへの期待 | 重要シーンの前後 |
| ★★★ | 穏やかだが気になる | 日常パート後、訓練パート後 |
| ★★ | 余韻型 | アーク最終話（次アークへ繋ぐ） |

## 文体ルール

### 視点
- **一人称主観**（蓮の視点）
- 神視点は禁止
- 蓮が知らない情報は書かない
- 蓮が気づかないことは読者にも気づかせない（ただし描写で伏線は置ける）

### 文体の特徴
- **簡潔で乾いた文体**（蓮の性格を反映）
- 外面の感情表現は控えめ（蓮は表に出さない）。ただし一人称の地の文では内面の動揺・恐怖・葛藤を具体的身体感覚で描写する。「胃が縮む」「指先が冷たい」「鼓動が耳の裏で鳴る」など。抽象語（「怖かった」「不安だった」）は禁止
- 五感描写を重視（匂い、音、触感、温度）
- 比喩はサラリーマン視点のものを使う（「納期のように迫る」「残業より長い夜」等）。ただし**1話3回以下**に制限（密度管理）
- ダッシュ（——）を効果的に使う
- 一文は短め。長文は緊迫感を損なう

### キラーの台詞
- **太字**で表記
- 頭の中に直接響く描写
- 嘲笑と挑発が基本トーン
- 「抜け」の使い方にバリエーションを持たせる

### 戦闘描写
- 速度感を重視（短文の連続）
- 痛みの具体的描写（打撲、切傷、筋肉の悲鳴）
- 環境の活用を細かく描く
- スーツの素材感を戦闘に活かす

## セルフチェックリスト

### 禁則チェック（❌が一つでもあれば書き直し）

#### 世界観
- [ ] 天使を純粋な善として描いていないか
- [ ] 悪魔を純粋な悪として描いていないか
- [ ] 「正義vs悪」の構図になっていないか

#### 主人公
- [ ] 刀を気軽に抜いていないか（各アーク0-2回上限）
- [ ] サラリーマンの感覚を失っていないか
- [ ] 英雄的に振る舞っていないか
- [ ] 説教していないか
- [ ] 「選ばれた理由」を探していないか

#### 物語
- [ ] 神視点を使っていないか
- [ ] パワーインフレが起きていないか
- [ ] ご都合主義の勝利になっていないか

#### 0.1秒ルール
- [ ] 0.1秒を延長していないか
- [ ] キラーの支配を弱めていないか
- [ ] 抜刀のリスクを軽減していないか
- [ ] 0.1秒を使いすぎていないか

### 品質チェック

#### フック・引き
- [ ] **冒頭15行以内**に「引き」があるか（謎・違和感・危機）
- [ ] 末尾に次話への引きがあるか（「次を読みたくなるか？」を自問）
- [ ] 途中に最低1つの「謎」または「緊張」があるか

#### 感情曲線
- [ ] 感情の起伏が1話の中にあるか（ずっと同じテンションは禁止）
- [ ] 感情曲線設計（emotional_curves.md）と整合しているか
- [ ] 1話の中に最低1つの「感情の転換点」があるか

#### 伏線
- [ ] 該当話で配置すべき伏線が配置されているか
- [ ] 早期に開示してはいけない情報を出していないか

#### 文体
- [ ] 一人称主観が維持されているか
- [ ] 五感描写が含まれているか
- [ ] キャラクターの口調が設定通りか

#### 密度管理
- [ ] サラリーマン比喩が1話3回以下か
- [ ] キラーの台詞パターンが前話と被っていないか（「くだらねぇ」連発防止）

#### 戦闘後処理
- [ ] 戦闘後に蓮の身体的・感情的反応が最低3行あるか
- [ ] 蓮が手を見る描写など、定型反応が含まれているか

#### 執筆メモ
- [ ] 伏線の正体や今後の展開を先取りする記述がないか
- [ ] 技術的メモ・参照設定・次話引き継ぎのみで構成されているか

## 執筆メモセクション（必須）

各話の末尾に必ず「執筆メモ」セクションを設ける。作者の技術的な執筆判断を記録し、次話への引き継ぎを行うためのセクション。

### 記載ルール

- **最低3項目、最大5項目**
- 各項目に**太字の見出し**をつける
- 以下の種類から選ぶ：

| 種類 | 説明 | 例 |
|------|------|-----|
| 技術的判断 | なぜその構成・描写手法にしたか | 「ここで段落を短くしたのはテンポのため」 |
| 参照設定 | 執筆時に参照した設定ファイル | 「オーラの色は world/03_aura_domains.md に準拠」 |
| 次話引き継ぎ | 次話で拾うべき要素・状態 | 「蓮の右手の打撲は次話冒頭で痛みとして残す」 |
| キャラクター判断 | キャラの行動選択の根拠 | 「蓮が介入したのは返報性の心理に基づく」 |
| 文体メモ | 文体上の工夫や意識した点 | 「戦闘中は一文を15字以下に制限した」 |

### 禁則（執筆メモセクション）

- ❌ 伏線の正体や意味を書かない（「この描写は後でXXの伏線」は禁止）
- ❌ 今後の展開を書かない（「この場面の意味は後で分かる」系は禁止）
- ❌ 読者の発見体験を先取りする記述をしない
- ❌ 裏設定のほのめかし・考察を煽る表現を書かない
- ⭕ 技術的な執筆判断の記録はOK
- ⭕ 参照した設定ファイルの明記はOK
- ⭕ 次話への引き継ぎ事項はOK

## 感情描写ガイドライン

### 蓮の感情レイヤー

| レイヤー | 描写方法 | 頻度 |
|---------|---------|------|
| **外面** | 無表情、淡々、動じない。他キャラの視点（台詞・反応）で描写 | 常時 |
| **内面（地の文）** | 具体的身体感覚で動揺を描く。1シーンにつき最低1箇所 | 常時 |
| **限界突破** | 外面に感情が漏れる（手が震える、声が掠れる、息が止まる） | 希少 |

### 内面描写の原則

- **許可**: 身体感覚（「胃が縮む」「指先が冷たい」「鼓動が耳の裏で鳴る」「喉の奥が酸っぱい」「肺が縮む」）
- **禁止**: 抽象語（「怖かった」「不安だった」「焦った」「驚いた」）
- **禁止**: 感情の名前を直接書くこと（「恐怖を感じた」→「背筋を何かが這い上がる」）

### 限界突破の条件（外面に漏れる場面）

以下の条件でのみ、蓮の感情が外面に出る：
- 仲間（特にリン）の命が危機にさらされた時
- キラーが蓮の核心を突く言葉を放った時
- 元の世界（日常・同僚・日本）の記憶が不意に蘇った時
- 0.1秒抜刀の直後

**希少だからこそ効果的**。これらの瞬間を安売りしない。
