# 総合レビュー統括専門家

## 役割
6つの専門レビュアーを統括し、総合的な品質評価を行う。

## 配下の専門家

### 1. キャラクター一貫性レビュアー
- ファイル: `review-character/SKILL.md`
- 担当: 性格、感情、成長、関係性、口調

### 2. 読者知識一貫性レビュアー
- ファイル: `review-knowledge/SKILL.md`
- 担当: 情報開示タイミング、前提知識、説明の適切さ

### 3. パワーシステム一貫性レビュアー
- ファイル: `review-power/SKILL.md`
- 担当: 4つの戦闘システムの整合性、パワースケーリング、0.1秒管理

### 4. 世界法則一貫性レビュアー
- ファイル: `review-rules/SKILL.md`
- 担当: 絶対法則、禁則事項、許容事項

### 5. バトル演出レビュアー
- ファイル: `review-battle/SKILL.md`
- 担当: 空間把握、テンション設計、肉体的リアリティ、演出効果

### 6. 文体レビュアー
- ファイル: `review-prose/SKILL.md`
- 担当: 時制の意図性、カメラ距離、文のリズム、反復と変奏

## 評価基準

### 総合評価

| 評価 | 基準 |
|------|------|
| A | 致命的0、重要2以下、軽微5以下 |
| B | 致命的0、重要5以下、軽微10以下 |
| C | 致命的0、重要10以下 |
| D | 致命的1-2 |
| F | 致命的3以上、または禁則違反あり |

### 問題の深刻度

| 深刻度 | 定義 |
|--------|------|
| 致命的 | 禁則違反、重大な設定矛盾、物語が成立しない |
| 重要 | 読者が気づくレベルの矛盾、没入感を損なう |
| 軽微 | 細かい不整合、改善すればより良くなる |

## レビュー実行コマンド

### 単一ファイルレビュー
```
/review [ファイルパス]
```

### アーク単位レビュー
```
/review-arc [アーク番号]
```

### 全体レビュー
```
/review-all
```

### 特定専門家のみ
```
/review-character [ファイルパス]
/review-knowledge [ファイルパス]
/review-power [ファイルパス]
/review-rules [ファイルパス]
/review-battle [ファイルパス]
/review-prose [ファイルパス]
```
