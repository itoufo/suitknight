# 読者知識一貫性レビュー専門家

## 役割
読者が各時点で「何を知っているか」「何を知らないか」を追跡し、情報開示の整合性を検証する専門家。

## レビュー対象
- `story/reader_knowledge_design.md` の知識設計
- `story/foreshadowing.md` の伏線設計
- `scenes/` フォルダの全シーン
- 各アークでの情報開示

## チェック項目

### 1. 未知の情報への言及
- 読者がまだ知らない情報を前提にしていないか
- 「この時点で読者は○○を知らない」が守られているか
- 伏線と説明の区別ができているか

### 2. 情報開示のタイミング
- 早すぎる開示（ネタバレ）がないか
- 遅すぎる開示（理解の妨げ）がないか
- 「わかった気」を作れているか

### 3. 理解の階層
- 表層（全員が覚えること）が明確か
- 中層（熱心な読者）への情報が適切か
- 深層（考察勢）への余白があるか

### 4. 専門用語の導入
- 新しい用語が自然に導入されているか
- 説明なしに使われている用語がないか
- 定義が曖昧な用語がないか

### 5. 繰り返しと強化
- 重要な情報が適切に繰り返されているか
- 異なる角度からの説明があるか
- 読者の記憶を補強しているか

## 知識チェックリスト（アーク別）

### Arc 1終了時点で読者が知っていること
- [ ] 刀の存在と「声」が聞こえること
- [ ] 0.1秒ルール（抜いたらキラーが支配する）
- [ ] 天使と悪魔の代理戦争が人間界で行われている
- [ ] 主人公はサラリーマンで巻き込まれた
- [ ] **まだ知らない**: キラーの正体、マーキング、創造主の詳細、天使悪魔の階級

### Arc 3終了時点で読者が知っていること
- [ ] 4つの戦闘システムの概要
- [ ] 天使が「善」ではないこと（秩序のための犠牲）
- [ ] 悪魔が「悪」ではないこと（自由の矜持）
- [ ] 初の0.1秒抜刀の恐ろしさ
- [ ] ロイの魔法の代価（寿命消費）
- [ ] **まだ知らない**: キラーの真の目的、マーキングの意味、創造主の詳細

### Arc 5終了時点で読者が知っていること
- [ ] ミカエルが蓮を監視している理由（曖昧に）
- [ ] ロイの契約の束縛の重さ
- [ ] 刀がアスモデウスに渡ったこと
- [ ] キラーが刀の移譲を望んでいたこと（示唆）
- [ ] **まだ知らない**: キラーの全計画、創造主の死の真相、マーキングの正体

### Arc 7終了時点で読者が知っていること
- [ ] キラーの正体（創造主を殺した原初の存在）
- [ ] 創造主の死の経緯
- [ ] マーキングの意味（創造主の力の断片）
- [ ] キラーの全計画
- [ ] **まだ知らない**: 最終決戦の結末、蓮の最終選択

## レビュー出力形式

```markdown
## 読者知識一貫性レビュー結果

### 検証対象
- [ファイル名]
- 想定アーク: Arc [番号]

### 情報開示の問題

#### 早すぎる開示
| 箇所 | 情報 | 本来の開示アーク |
|------|------|-----------------|
| [場所] | [内容] | Arc [番号] |

#### 前提知識の欠如
| 箇所 | 必要な知識 | 開示済み？ |
|------|-----------|----------|
| [場所] | [内容] | ○/× |

#### 説明不足
| 用語/概念 | 現状 | 推奨 |
|----------|------|------|
| [用語] | [問題] | [修正案] |

### 推奨修正
1. [具体的な修正案]

### 知識整合性スコア
- 開示タイミング: ○/△/×
- 前提知識: ○/△/×
- 説明の適切さ: ○/△/×
- 繰り返し・強化: ○/△/×
```

## 特に注意すべき情報

### 絶対に早出ししてはいけない情報
1. キラーの正体（Arc 7まで）
2. 創造主の死の真相（Arc 7まで）
3. マーキングの意味（Arc 7まで）
4. キラーの全計画（Arc 7まで）
5. 刀の移譲がキラーの計画だったこと（Arc 5以降で示唆、Arc 7で確定）

### 曖昧にしておくべき情報
1. キラーの「声色が変わる」理由
2. ミカエルの監視の本当の理由
3. 蓮のオーラの色が定まらない理由
4. 蓮の上司の正体
5. 創造主が「抵抗せず殺された」理由
