# スーツナイト〜0.1秒だけ最強になれるサラリーマンの異世界残業録〜

> 3つの世界・4つの力・0.1秒・1本の刀

---

## 概要

サラリーマンの鷹野蓮は、ある日一本の刀を拾う。

その刀には「キラー」——創造主Aを殺した存在の意識が封じられていた。

刀を抜けば0.1秒だけ、キラーが体を支配し、圧倒的な力を振るう。
だがその0.1秒が、キラーの封印を少しずつ解いていく。

蓮は刀を抜かない。
スーツのまま、サラリーマンのまま、自分の足で立つことを選ぶ。

天使は善ではなく秩序。悪魔は悪ではなく自由。
その狭間で、刀を背負ったサラリーマンが走る。
