# キラー（刀に封じられた存在）

## 基本情報

| 項目 | 設定 |
|------|------|
| 正体 | 原初の存在（創造主が最初に創った存在） |
| 現在の形態 | 刀に封じられた声（姿は見えない） |
| 性格 | 冷徹・合理的・嘲笑的 |
| 目的 | 封印の完全解除→創造主の残存力の完全吸収 |
| 口癖 | 「抜け」 |

---

## 人格

- 知性は創造主に次ぐ（自称）
- 冷徹で合理的。感情的な判断をしない
- 嘲笑的——蓮の「抜かない」を嘲る
- 嘘はつかない（ただし真実を隠すことはある）
- 主人公を名前で呼ばない（「おまえ」「サラリーマン」）
- しかし時折、声色が変わる瞬間がある（理由はArc 7で判明）

---

## 話し方のパターン

### 平常時
```
「抜けよ、サラリーマン。退屈で死にそうだ」
「おまえの我流とやらは、見てるこっちが恥ずかしいぞ」
「0.1秒でいい。それだけで全部終わる」
「俺を使え。使わないなら、おまえに価値はない」
```

### 戦闘時（蓮が苦戦中）
```
「まだ抜かないのか。馬鹿だな、おまえは」
「死ぬぞ。本当に死ぬぞ、サラリーマン」
「おまえが抜かないなら、次は見殺しだ」
「……抜け。今すぐ抜け」（声色が変わる）
```

### 主人公が本当に危機の時
```
「……おい」（声が低くなる）
「死ぬな」（命令口調だが、何か違う響き）
「抜け。頼むから、抜け」（「頼む」を使う異常事態）
```

### 0.1秒支配中
```
（笑い声）
「ハハハハハハハ！ 0.1秒——たった0.1秒だ！」
「この身体、借りるぞ」
「……足りない。0.1秒じゃ足りない」
```

## 各フェーズの目的

| フェーズ | 目的 | 蓮への態度 |
|---------|------|-----------|
| 育成期（Arc 1-2） | マーキングの成長を促す | 嘲笑と挑発 |
| 危機誘発期（Arc 3-4） | 抜刀を強いる状況を作る | 焦りと苛立ち |
| 移譲期（Arc 5） | 刀をアスモデウスに渡す | 沈黙（計画通り） |
| 解放期（Arc 6-8） | 封印解除の最終段階 | 困惑→興味→??? |

---

## 主人公との関係の変化

1. **道具**（Arc 1）：蓮はマーキングを持つ便利な器
2. **苛立ち**（Arc 2-3）：なぜ抜かない？理解できない
3. **困惑**（Arc 4-5）：蓮の頑固さは計算外
4. **興味**（Arc 6-7）：蓮の選択に、かつての自分を見る
5. **???**（Arc 8）：最終的な関係は物語の核心

---

## 0.1秒支配中の描写ルール

- 蓮の体を完全に支配する（蓮の意識はあるが動けない）
- 漆黒のオーラが噴出する
- 超高速の動き（常人には見えない）
- 圧倒的な破壊力（地形が変わるレベル）
- 笑い声が響く
- 0.1秒後、自動的に鞘に戻る
- 蓮の体に甚大な負荷（筋繊維断裂、骨のひび、内臓出血）

---

## 禁則

- ❌ 嘘をつかない（真実を隠すことはある）
- ❌ 主人公に同情しない（と見せかけて…）
- ❌ 封印解除の真の条件を明かさない
- ❌ 単純な悪として描かない（合理的で打算的）
- ❌ 主人公を名前で呼ばない
