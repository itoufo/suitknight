# 白峰 玲央（しらみね れお）/ ロイ — ライバル・魔法使い

## 基本情報
| 項目 | 設定 |
|------|------|
| 名前 | 白峰 玲央（しらみね れお）、通称ロイ |
| 年齢 | 26歳 |
| 所属 | 独立派→天使派（ミカエル契約後） |
| 武器 | 魔導杖「叡智の書架」＋グリモワール |
| 戦闘力 | C→A（ミカエル契約後） |

## 性格
- 知的、冷静、プライドが高い
- 天才的な魔法使い。努力も惜しまない
- 蓮の「抜かない」に激怒する理由：力があるのに使わない＝力を持つ者への侮辱
- ミカエル契約後、秩序と自由意志の間で葛藤する

## ミカエルとの契約
- 寿命消費の代わりに天使の力を代価にできる
- 代償：ミカエルの命令に絶対服従
- ロイにとっては「力を得るための合理的選択」だった
- しかしArc 5でミカエルの命令と自分の良心が対立する

## 蓮との対比
| ロイ | 蓮 |
|------|-----|
| 力を最大限使う | 力を使わない |
| 天才 | 凡人 |
| 契約で力を得る | 我流で鍛える |
| 自由を売った | 自由を守っている |

## 成長の軌跡
1. 天才的魔法使い（Arc 2で登場）
2. ミカエル契約（Arc 3）→強大な力を得る
3. 秩序への服従に苦しむ（Arc 4-5）
4. ミカエルの命令で蓮を裏切る（Arc 5）→自己嫌悪
5. 最終的に自分の意志を選ぶ（Arc 6-7）
