# Aldric（アルドリック）— 聖騎士団長

## 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | Aldric（アルドリック） |
| 年齢 | 63歳 |
| 役職 | 聖騎士団長（教会最高指揮官） |
| 契約天使 | ミカエル（間接契約。直接ではない） |
| 武器 | 大盾と戦槌 |
| 所属 | 教会 |

## 性格

### 一言定義
> 「秩序に従い続けた人間の末路」

### 詳細
- かつては志のある人間だった。「天使の力で人々を守る」という信念があった
- しかし数十年の契約で**意志が摩耗**。天使の命令に従うだけの存在に
- 目に光がない。声に感情がない。だが時折、昔の自分が顔を覗かせる
- 天使の傀儡に限りなく近いが、完全な傀儡ではない——その「残り滓」が悲劇

### 対人関係
- ミカエルの命令を絶対とする。疑問を持つ力が残っていない
- シグルドを後継者として認めているが、シグルドの「人間性」を危険だとも感じている
- セドリックを「理想的な部下」と評価（＝自分と同じ道を歩んでいると気づかない）
- 若い騎士たちの名前を時々間違える（個人を認識する力が衰えている）

## 戦闘スタイル
- 全盛期はミカエルに最も近い戦闘スタイルだった
- 今は衰えたが、経験と技術で中位悪魔程度なら圧倒
- オーラは灰白色。かつては純白だったが、くすんでいる
- 大盾で味方を守り、戦槌で敵を打つ——「守護者」の戦い方

## 物語における役割
- **蓮が「刀を抜かない」ことで守っているものが、ここにある**
- 秩序に従い続けた結果、自分の意志を失った人間の姿
- 0.1秒の誘惑に「従うこと」と「自分の意志を捨てること」の恐怖を可視化
- シグルドにとっての「未来の自分」の恐怖

## テーマとの関係
> **「選び続けなかった者の末路」**
- 蓮は刀を抜かず、自分の足で立ち続ける
- アルドリックは天使の力に従い続け、自分の足で立つことをやめた
- 両者は鏡像。蓮がいつか到達しうる「もう一つの結末」

## 話し方
- 虚ろ。定型文が多い。時折、昔の口調が漏れる
- 「……ミカエル様の御意志のままに」
- 「シグルド……副団長。お前は……いや、命令を伝達する」
- （昔の口調が漏れる瞬間）「守る……そうだ、俺は守るために……」
