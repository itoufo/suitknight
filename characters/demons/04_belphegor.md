# ベルフェゴール（怠惰）— 戦略担当

## 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | ベルフェゴール |
| 対応する罪 | 怠惰 |
| 序列 | 第六位 |
| 領域 | 「停滞の領域」— 範囲内の全存在の行動速度を極端に低下させる |
| 武器 | 扇「無為（アパティア）」— 振ると対象の「やる気」を奪う |

## 性格

- 最小の力で最大の効果を狙う**極度の合理主義者**
- 「怠惰」は「サボり」ではなく「無駄を徹底的に排除した結果」
- アスモデウスの戦闘狂を「非効率」と嘆くが、その自由さを内心羨む
- ザガンの交渉術を高く評価。悪魔側の実質的な頭脳
- 常に眠そう。だが思考は七大悪魔で最も鋭い

## 戦闘能力

- 直接戦闘は好まないが、領域は極めて厄介
- 「停滞の領域」で敵の動きを遅くし、味方が一方的に攻撃する状況を作る
- 扇「無為」は精神攻撃——戦意を奪い、戦闘を放棄させる
- 本気を出せばアスモデウスすら手こずるが、「本気を出すのが面倒」

## 物語における役割

- 悪魔側の戦略を立案する頭脳
- 蓮の「刀を抜かない」合理性に興味を持つ——「無駄を省く」という点で共鳴
- ただし蓮の動機は「合理性」ではなく「意志」——その違いに気づけるか
- 雷牙の契約者として、元教会の人間を悪魔側に取り込んだ張本人

## テーマとの関係

> **「最も効率的な自由は、何もしないこと」——だがそれは本当に自由か**

- ベルフェゴールの「怠惰」は究極の効率追求
- しかし「何もしない自由」は「何もできない不自由」と紙一重
- 蓮の泥臭い努力を「非効率」と断じるが、その非効率さに惹かれる

## 話し方

- 気怠げ。あくびを挟む。だが核心を突く
- 「……めんどくさい。アスモデウス、また暴れてるのか」
- 「ザガン、お前に任せる。俺は寝てる」
- 「サラリーマン。刀を抜かないのは合理的だ。……だが理由が違うな」
- 「本気？ 出したことないな。出す必要がなかったから」
