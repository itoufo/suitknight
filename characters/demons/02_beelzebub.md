# ベルゼブブ（暴食）— 第二位

## 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | ベルゼブブ |
| 対応する罪 | 暴食 |
| 序列 | 第二位 |
| 領域 | 「飢餓の領域」— 範囲内のあらゆるエネルギーを吸収する |
| 武器 | なし（自身の体が武器） |

## 性格

- 寡黙で巨大。常に飢えている——が、飢えているのは「食」ではなく**「意味」**
- 自由に生き、自由に喰らい、自由に在る。だが「それで何になる？」という虚無を抱える
- アスモデウスが唯一「対等」と認める存在。互いに敬意を持つ
- 言葉少なだが、発言には重みがある

## 戦闘能力

- **力の吸収**に長ける。敵のオーラ、領域、魔法——あらゆるエネルギーを喰らう
- 物理的にも巨大で、単純な腕力でも大悪魔クラスを圧倒
- 領域「飢餓の領域」は受動的——攻めるのではなく、**相手の力を枯渇させる**
- 長期戦では無敵に近いが、一撃必殺の爆発力には欠ける

## 物語における役割

- Arc 5-6：アスモデウスがキラーの刀に支配された際、**蓮と一時共闘**
- アスモデウスを止められるのはベルゼブブだけ——その事実が共闘の根拠
- 蓮の「意味がなくても立ち続ける」姿に、自身の虚無への回答を見るかもしれない

## テーマとの関係

> **「自由であることの虚しさ」**

- 自由に何でもできる。だが何をしても満たされない
- 「暴食」は身体の飢えではなく、存在の飢え
- 蓮の「意味がなくても選ぶ」が、ベルゼブブの問いへの一つの答え

## 話し方

- 極めて寡黙。一言で核心を突く
- 「……腹が減った」（口癖。だが食事ではない何かへの飢え）
- 「アスモデウス。お前を止めるのは、俺の役目だ」
- 「サラリーマン。お前は……何を喰って生きている？」
