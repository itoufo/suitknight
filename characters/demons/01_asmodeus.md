# アスモデウス — 最強の悪魔（七大悪魔の筆頭）

## 基本情報
| 項目 | 設定 |
|------|------|
| 名前 | アスモデウス |
| 階級 | 七大悪魔の筆頭 |
| 武器 | 大剣「自由の翼（リベルタス）」完全解放 |
| 領域 | 自由領域（あらゆる制約を解除） |
| 戦闘力 | Sランク（ミカエルと互角） |

## 性格
- 自由を愛する高潔な悪魔
- 部下を駒とは見ない。一人一人の自由を尊重する
- 戦闘を心から楽しむ
- 強い相手との戦いに歓喜する
- 卑怯な手段を使わない（自由の矜持）

## 刀を受け取る経緯（Arc 5）
1. 凛が重傷を負う
2. 蓮は凛を守るため、戦闘を放棄し刀を手放す決断をする
3. キラーが意図的にアスモデウスを引き寄せる
4. アスモデウスが刀を拾う（強大な力への興味）
5. 戦闘を楽しむ性格のため、何度も抜刀する
6. キラーの封印がどんどん緩む

## アスモデウスの悲劇
- 自由の体現者が、キラーに操られていく
- 最初は自分の意志で抜刀していた
- 次第にキラーの声に誘導されるようになる
- 「自由」を掲げる者が「操り人形」になる皮肉
- しかし完全に支配される前に、蓮との最終対決
