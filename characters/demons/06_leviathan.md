# レヴィアタン（嫉妬）— 海の支配者

## 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | レヴィアタン |
| 対応する罪 | 嫉妬 |
| 序列 | 第五位 |
| 領域 | 「深淵の領域」— 海を自在に操り、領域内を深海に変える |
| 武器 | なし（海そのものが武器） |

## 性格

- エルダの海域を支配。陸の戦争には基本的に関与しない
- 穏やかで物静か。七大悪魔の中で最も「話が通じる」ように見える
- しかし他の七大悪魔が「陸」で活躍することへの**静かな嫉妬**を抱える
- 一度怒ると最も危険。海ごと敵を飲み込む
- 「関わらないこと」が自分の自由だと言い聞かせている

## 戦闘能力

- 海域では無敵。陸の戦闘力は大幅に低下する
- 「深淵の領域」を展開すると周囲が深海に変わる——陸でも海戦を強制
- 海流、水圧、津波——自然災害レベルの攻撃
- 防御面でも水の壁は極めて頑強

## 物語における役割

- エルダの海域という「もう一つの戦場」の支配者
- 陸の戦争に関与しないことで独自の立場を保つ
- 蓮が海域に関わるエピソードで登場
- 「関わらない自由」を体現しつつ、本心では関わりたい矛盾

## テーマとの関係

> **「関わらないことは自由なのか、孤立なのか」**

- 蓮は巻き込まれながらも「自分で選ぶ」
- レヴィアタンは自ら「関わらない」を選んだ——だがそれは本当に「選んだ」のか
- 嫉妬の本質：他者が持つものを欲しながら、手を伸ばさない

## 話し方

- 静かで流れるような口調。水のイメージ
- 「陸の争いに、海は関知しない」
- 「嫉妬？ ……そうかもしれない。陸は賑やかで、いいな」
- 「私の海に入るのなら——覚悟はできていますか」
- 「アスモデウスは自由だと言う。私は……自由なのだろうか」
