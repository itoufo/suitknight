# サタン（憤怒）— 純粋な破壊衝動

## 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | サタン |
| 対応する罪 | 憤怒 |
| 序列 | 第四位 |
| 領域 | 「憤怒の領域」— 範囲内の全存在の攻撃力を際限なく増幅する（敵味方無差別） |
| 武器 | 拳のみ。武器を持たない唯一の七大悪魔 |

## 性格

- 七大悪魔で最も「悪魔らしい」。暴力的で破壊的で衝動的
- だが彼の怒りには理由がある
- **創造主Aが死んだ時、世界の崩壊を目撃した最初の悪魔**
- その衝撃が永遠の怒りに変わった——怒り続けることでしか崩壊の悲しみを処理できない
- アスモデウスとは犬猿の仲。「自由」を語るアスモデウスを「偽善」と罵る

## 戦闘能力

- **純粋な戦闘力はアスモデウスに匹敵**。正面からの殴り合いなら互角
- 武器を持たず、拳と蹴りだけで戦う。技術ではなく圧倒的な力
- 「憤怒の領域」は味方も強化するが敵も強化する——純粋な力比べの空間
- 怒りが増すほど強くなる。理論上の上限がない（が、体が持たない）

## 物語における役割

- アスモデウスの対極として「自由の暗部」を体現
- 「自由に怒る」ことは自由か、それとも怒りの奴隷か
- 創造主Aの世界崩壊の目撃者として、物語の深層に関わる情報を持つ
- ただし感情的すぎて冷静に語ることができない

## テーマとの関係

> **「怒り続けることでしか悲しみを処理できない」**

- サタンの怒りは「選択」ではない。悲しみから逃れるための反射
- 「選ばない」の別形態——感情に突き動かされ、自分の意志で在ることができない
- 蓮の冷静さ（サラリーマンの冷静さ）がサタンを最も苛立たせる

## 話し方

- 怒鳴る。叫ぶ。言葉より拳が先に出る
- 「うるせぇ！！ 自由だ何だと——お前に何がわかる！！」
- 「アスモデウス……てめぇの『自由』は、ただの逃げだ」
- 「壊す。全部壊す。壊さなきゃ……壊さなきゃ、俺は……」
- （稀に静かになる）「……あの時、世界が消えた。全部。一瞬で」
