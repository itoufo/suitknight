# マモン（強欲）— 富と取引の支配者

## 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | マモン |
| 対応する罪 | 強欲 |
| 序列 | 第三位 |
| 領域 | 「契約の領域」— 範囲内での約束・取引を絶対化する |
| 武器 | 天秤「等価（エクイヴァレンス）」— 取引の対価を強制する |

## 性格

- あらゆるものを「取引」の対象と見る。命、魂、感情——全てに値段がつく
- 戦闘より交渉と策略。直接戦うことを好まない
- 人間を「価値ある商品」と見る。悪意はないが善意もない
- ザガンの本来の上司。ギルドの経済基盤を支える黒幕
- 穏やかで紳士的。だが「取引」から逃れることは許さない

## 戦闘能力

- 直接戦闘力は七大悪魔で最も低い
- だが「契約の領域」内では無敵——取引を持ちかけ、拒否した者に対価を強制
- 天秤「等価」は相手の「最も価値あるもの」を奪う
- 交渉術そのものが武器。戦わずして勝つ

## 物語における役割

- ギルドの経済と物資を掌握する黒幕
- ザガンを通じて人間社会に影響力を行使
- 蓮に「取引」を持ちかける——「刀を渡せば、元の世界に帰してやる」
- 蓮の拒絶が「取引できない存在」としてマモンを困惑させる

## テーマとの関係

> **「自由＝選択肢を持つこと」という解釈。選択肢を買い集める**

- マモンにとって自由とは「何でも手に入る状態」
- しかし蓮は「手に入れること」ではなく「自分で選ぶこと」に価値を置く
- 「すべてを持つ者」と「何も持たないが選ぶ者」の対比

## 話し方

- 紳士的で丁寧。商人の話し方
- 「さて、取引といきましょう。あなたの望みは？」
- 「対価なき要求は、強欲ですらない。ただの無礼です」
- 「ザガン。お前の交渉は甘い。だが嫌いではない」
- 「帰りたいのでしょう？ 元の世界に。対価は——その刀です」
