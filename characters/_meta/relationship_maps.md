# 全陣営 関係性マップ

## 天使陣営

```
ミカエル（統括）
├─ガブリエル（参謀）…情報で支える。ミカエルの判断を論理で補佐
├─ラファエル（治癒）…唯一意見できる。「慈悲的秩序」
├─ウリエル（粛清）…忠実な剣。だがミカエルの迷いを危険視
│
├─セドリック（人間幹部）…絶対忠誠。疑問を持たない
├─ロイ（ミカエル直契約）…道具。才能ゆえの特別扱い
├─シグルド（ラファエル契約・副団長）…人間性を保つエリート
├─ミリアム（聖騎士）…秩序を信じる一般兵
└─アルドリック（聖騎士団長・傀儡）…秩序に従い続けた末路
```

### 天使陣営・対立軸

| # | 対立 | 内容 |
|---|------|------|
| 1 | ミカエル vs ウリエル | 揺らぐ指導者 vs 原理主義者。ウリエルがミカエルの「迷い」を粛清対象と見なし始める |
| 2 | ラファエル vs ウリエル | 慈悲 vs 冷酷。天使の中の路線対立 |
| 3 | ロイ vs ミリアム | 特別 vs 普通。同期でありながらミカエル直契約で引き離された |
| 4 | シグルド vs セドリック | 人間性 vs 忠誠。教会内改革の芽 vs 完全な従属 |
| 5 | アルドリックの存在 | 秩序に従い続けた人間の行く末。シグルドの「未来の姿」 |

### 天使陣営・主要関係

- **ミカエル → ロイ**: 道具として契約。才能は認めるが対等ではない
- **ミカエル → ラファエル**: 意見を聞く唯一の相手。だが最終判断は自分
- **ウリエル → ミカエル**: 秩序への忠誠であり、ミカエル個人への忠誠ではない
- **ラファエル → シグルド**: 契約者。シグルドの人間性に興味を持ち始めている
- **セドリック → ミリアム**: 上官と部下。セドリックの冷酷さにミリアムは時折疑問を抱く
- **ミリアム → ロイ**: 同期。才能差への嫉妬と尊敬。取り残された感覚
- **ミリアム → シグルド**: 「理想の上官」として慕う

## 悪魔陣営

```
アスモデウス（筆頭・自由）
├─ベルゼブブ（暴食・虚無）…唯一の対等。意味を問う
├─マモン（強欲・取引）…経済的黒幕。自由＝選択肢
├─サタン（憤怒・破壊）…犬猿の仲。怒りの裏に悲しみ
├─レヴィアタン（嫉妬・海）…孤立者。関わらない自由
├─ベルフェゴール（怠惰・戦略）…頭脳。合理で自由を支える
├─ルシファー（傲慢・消息不明）…Arc 7-8で登場。核心に関わる
│
├─ザガン（交渉人）…穏健派外交官。マモンの部下
├─リル（下位悪魔）…善良な悪魔
├─ヴォルク（ギルドマスター・ベルゼブブ契約）…自由を守るための不自由
├─ラグナ（赤旗・ベルフェゴール契約）…自由の代価を払った男
└─ナディア（情報屋）…誰も信じない少女。ザガンに恩義
```

### 悪魔陣営・対立軸

| # | 対立 | 内容 |
|---|------|------|
| 1 | アスモデウス vs サタン | 自由 vs 怒り。悪魔の二面性 |
| 2 | アスモデウス vs ベルフェゴール | 本能 vs 合理。自由の解釈の違い |
| 3 | ベルゼブブの虚無 | 自由に意味はあるのか？ |
| 4 | マモンの支配 | 取引は本当に自由か？ 経済で縛る自由 |
| 5 | レヴィアタンの孤立 | 関わらないのは自由か孤独か |
| 6 | ヴォルク vs ラグナ | 組織の自由 vs 個人の自由。元傭兵団の因縁 |
| 7 | ナディアの不信 | 信じないことは自由か？ |

### 悪魔陣営・主要関係

- **アスモデウス ↔ ベルゼブブ**: 唯一の対等。無言の信頼
- **アスモデウス ↔ サタン**: 犬猿の仲。互いの自由の解釈が衝突
- **マモン → ザガン**: 上司と部下。ザガンの交渉術を活用
- **ベルフェゴール → ラグナ**: 契約者。怠惰な悪魔と豪快な人間の奇妙なコンビ
- **ベルゼブブ → ヴォルク**: 契約者。寡黙な者同士、言葉少なく通じ合う
- **ザガン → ナディア**: 拾い親と拾われ子。ナディア唯一の恩人
- **ラグナ → ナディア**: 「危なっかしいガキ」と心配（本人には言わない）
- **ヴォルク → ラグナ**: 元傭兵団を吸収した因縁。複雑な感情

## 独立派

```
ヴェルナー（リーダー）
├─クルト（右腕・評議員）…現実主義。次期リーダー候補
├─リン（遊撃隊中核）…戦闘力の要。復讐心あり
├─エマ（医師）…非戦闘員の柱。リンの心の支え
├─カイ（遊撃隊員）…少年兵。リンの弟分
└─鷹野蓮（新参者）…異世界からの来訪者
```

### 独立派・対立軸

| # | 対立 | 内容 |
|---|------|------|
| 1 | 後継者問題 | ヴェルナー→クルトの不安。器の問題 |
| 2 | 合流派 vs 独立維持派 | クルトは合流も視野。リンは絶対反対 |
| 3 | リンの復讐心 vs 中立方針 | 個人の感情と組織の方針の衝突 |
| 4 | カイの未来 | 少年兵は戦い続けるべきか |
| 5 | 蓮の「なぜ戦う？」 | 全員を揺さぶる根本的な問い |

### 独立派・主要関係

- **ヴェルナー → クルト**: 信頼する右腕。後継者として期待
- **クルト → 蓮**: 当初反発。蓮の「理想主義」にイラつくが、次第に影響を受ける
- **リン → カイ**: 弟分。無意識に死んだ弟と重ねている
- **リン ↔ エマ**: 互いの支え。エマはリンの暴走のストッパー
- **エマ → 蓮**: 「普通の感覚」を持つ同志。非戦闘員の連帯
- **カイ → 蓮**: 「普通」を教えてくれた人。別の生き方の可能性
- **カイ → リン**: 憧れの「姐さん」。リンのように強くなりたい

## 陣営間の関係

### 天使 ↔ 悪魔
- **ミカエル ↔ アスモデウス**: 宿敵。秩序 vs 自由の頂点同士
- **ウリエル → 悪魔全体**: 殲滅対象。容赦なし
- **ラファエル ↔ ベルフェゴール**: 合理主義者同士。水面下で接触の可能性
- **マモン ↔ 教会**: 裏で取引。経済戦争

### 天使 ↔ 独立派
- **セドリック → 蓮**: 「異物」として排除対象
- **ウリエル → 蓮**: 「排除すべき異物」と断定
- **ミリアム → 蓮**: 蓮の「選ぶ」姿に動揺
- **シグルド → 独立派**: 対話の余地を探る（教会内では異端的姿勢）
- **ロイ ↔ 蓮**: ライバル関係。道具として使われるロイ vs 自分で選ぶ蓮

### 悪魔 ↔ 独立派
- **ラグナ ↔ リン**: 独立派時代に面識。ギルド勧誘の過去
- **ナディア → 蓮**: 蓮の「普通」に困惑。「なぜ戦う？」への「他に何があるの？」
- **ベルフェゴール → 蓮**: 刀を抜かない合理性に興味
- **ザガン → 独立派**: 穏健派として対話を試みる
